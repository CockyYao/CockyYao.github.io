RocketMq事务消息

发表于 2020-08-22 更新于 2020-11-23 分类于 Java

官方文档说明：

  • half message

指的是暂不能投递的消息，发送方已经将消息成功发送到了MQ服务端，但是服务端未收到生产者对于该消息的二次确认，这种情况称为半消息。

  • 消息回查

由于网络闪断、生产者应用重启等原因，导致某条事务消息的二次确认丢失，MQ服务端会主动向生产者发起询问，消息回查本地事务的执行状态。
